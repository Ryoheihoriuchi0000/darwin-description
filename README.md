# プライバシーポリシー

Darwin（以下、「本サービス」）は、ユーザーの個人情報を以下の通り取り扱います。

## 1. 取得する情報

- メールアドレス
- 表示名
- 端末情報
- 利用状況（アクセスログ等）
- Apple Sign In利用時のApple ID情報
- キャラクター・生息地等のゲームデータ

## 2. 利用目的

- サービスの提供および運営
- サービスの改善・開発
- 不正利用の防止
- お問い合わせへの対応

## 3. 第三者提供

以下のサービスを利用する場合があります：

- Firebase（Google LLC）
- Google Analytics

これらの提供元にデータが送信される場合があります。各サービスのプライバシーポリシーをご確認ください。

## 4. 保存期間

サービス提供に必要な期間のみ保存します。アカウント削除時は関連データを削除します。

## 5. データの削除

ユーザーは設定画面よりアカウントを削除することができます。削除後はすべての個人情報およびゲームデータが消去されます。

## 6. 安全管理

適切なセキュリティ対策を講じ、個人情報の漏洩・紛失・改ざんを防止します。

## 7. 子供のプライバシー

本サービスは13歳未満のお子様を対象としていません。13歳未満の方の個人情報を意図的に収集することはありません。

## 8. プッシュ通知

本アプリでは、プッシュ通知の配信のためにFirebase Cloud Messaging（FCM）を使用しています。通知の配信に必要なデバイストークンを収集・保存します。デバイストークンは通知の配信目的以外には使用しません。プッシュ通知はアプリ内の設定からいつでもオフにできます。

## 9. チャット機能

本アプリでは、チーム内のコミュニケーションのためにチャット機能を提供しています。送信されたメッセージはサーバー（Firebase Firestore）に保存されます。メッセージには送信者の表示名およびキャラクター情報が含まれます。

## 10. ブロック・報告機能

ユーザーは不適切なメッセージの報告およびユーザーのブロックを行うことができます。報告内容（報告理由、該当メッセージ）はサーバーに保存され、運営がサービス改善および安全性確保の目的で確認します。ブロックされたユーザーはチームから脱退し、ブロックした理由が通知されます。
